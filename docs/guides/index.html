Communication & Help

Mailing lists

Discussions take place in several mailing lists:

  • security-notices@opencast.org – Security announcements (low frequency)
  • announcements@opencast.org – Community announcements (low frequency)
  • users@opencast.org – Users list
  • dev@opencast.org – Developers list
  • anwender@opencast.org – German speaking community list
You can subscribe via Google Groups (linked above) or simply by sending a mail to [list]+subscribe@opencast.org (e.g. users+subscribe@opencast.org).

IRC channel (Chat)

For occasional quick questions we have an IRC channel on Freenode: #opencast.

Technical meeting

Our weekly meeting of developers and dev-ops takes place in BigBlueButton (room 1, password: welcome) on every Tuesday at 3pm UTC. Feel free to join!

Adopters meeting

Every last Wednesday of each month adopters meet in BigBlueButton (room 1, password: welcome). The meeting alternates between the international adopters meeting (at 3pm UTC) and the German speaking community meeting (2pm Europe/Berlin timezone). Watch the users list for announcements and feel free to join!

Development

Code

You can find the codebase of Opencast in the Git repository hosted at github.com/opencast/opencast. Feel free to fork to project and open pull requests. If you need help or more information, please do not hesitate to ask any question on the developers list.

Bugs

Please report security related bugs by sending an email to security@opencast.org.

Everything else should be filed as a ticket in our issue tracker.
